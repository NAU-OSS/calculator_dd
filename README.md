# Dandan Shao Open Source Software Dev

## Project Overview

This project is an open source Python calculator library designed to demonstrate basic arithmetic operations and automated testing using Python's unittest framework.


## Features

- Addition
- Subtraction
- Modular Python code structure
- Automated unit tests
- GitHub Actions CI integration


## Project Structure

src/
  calculator.py

tests/
  test_calculator.py

## Installation

Clone the repository:

git clone https://github.com/OSS-Doorway-Dev/dandan-shao-open-source-software-dev.git

Navigate to the project directory:

cd dandan-shao-open-source-software-dev

## Usage

Example:

```python
from src.calculator import add

result = add(2,3)
print(result)
```

## Running Tests

Run the test suite:
```python
python -m unittest discover -s tests
```
